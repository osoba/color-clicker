# Color Clicker Game

## Demo Live
Demo at https://osoba.github.io/color-clicker/

## Clone this repo
```
git clone https://github.com/osoba/color-clicker.git
cd color-clicker
```

## Run on your local machine
```
npm install
npm run dev
```

## Build for production and publish to github-pages
```
npm run build
npm run publish
```
